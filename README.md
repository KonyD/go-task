# go-task
CLI task manager written in go.

**Commands:**
+`add`    Adds a new task to your task list.
+`do`     Marks a task as completed.
+`help`   Help about any command.
+`list`   Lists all of your commands.

**Flags:**
`-h`, `--help`   help for task

**Building the project:**
```bash
go build main.go
```
